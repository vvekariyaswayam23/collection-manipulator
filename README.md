# 🎓 Student Data Organizer

> A simple and beginner-friendly **Python Console Application** to manage student records using **Lists, Dictionaries, Tuples, Sets, and Match-Case**.

## 📖 About Project

The **Student Data Organizer** is a Python-based command-line application that helps users manage student information efficiently.

It demonstrates Python fundamentals while providing an interactive menu-driven interface.

This project is perfect for:
- 🎓 Python Beginners
- 💻 College Students
- 📚 Practice of Python Data Structures
- 🚀 GitHub Portfolio Projects

---

## ✨ Features

✅ Add New Student

✅ Display All Students

✅ Update Student Information

✅ Delete Student

✅ Display All Subjects Offered

✅ Store Student DOB

✅ Uses Match-Case Statement

✅ Interactive Console Menu

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/student-data-organizer.git
```

### Go to Project Folder

```bash
cd student-data-organizer
```

### Run Program

```bash
python "student data(4).py"
```
